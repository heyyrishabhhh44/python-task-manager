# Python Task Manager

A simple Python-based task management and reporting system designed to simulate basic team workflows.

## Features
- Assign tasks to employees
- Track task status (Pending / Completed)
- View stored task data
- Generate summary reports

## Technologies Used
- Python
- CSV File Handling

## Project Objective
This project was built to practice modular Python development, file handling, and basic reporting logic similar to small ERP systems.

## How to Run
1. Ensure Python is installed
2. Navigate to the project folder
3. Run: python main.py
